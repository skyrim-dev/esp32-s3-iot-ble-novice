# 网络层 (Network Layer)

## 层级说明

网络层负责：
- **连接管理** - 网络连接建立和维护
- **协议接口** - TCP/IP、WiFi 等协议栈接口
- **事件处理** - 网络状态回调和事件通知
- **资源管理** - 网络相关资源分配和释放

## 目录结构

```
network/
├── CMakeLists.txt  # 组件配置
└── wifi/           # WiFi 连接模块
    ├── wifi.h
    └── wifi.c
```

## 设计原则

| 原则 | 说明 |
|------|------|
| 单一职责 | 只负责网络连接，不处理业务逻辑 |
| 状态通知 | 通过信号量/事件机制通知上层 |
| 自动重连 | 断开时自动尝试重连 |

```

                            ↓
```
