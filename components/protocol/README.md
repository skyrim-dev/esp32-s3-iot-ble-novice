# 协议层 (Protocol Layer)

## 层级说明

协议层负责：
- **协议实现** - MQTT、HTTP、CoAP 等通信协议
- **数据编解码** - JSON、Protobuf 等数据格式
- **平台对接** - 云平台通信和认证
- **消息封装** - 请求/响应的协议封装

## 目录结构

```
protocol/
├── CMakeLists.txt       # 组件配置
└── hw_iot_mqtt/        # 华为云 IoT MQTT 模块
    ├── CMakeLists.txt
    ├── *_config.*      # 协议配置和连接
    ├── *_topic.*       # Topic 定义和解析
    └── *_json.*        # JSON 编解码
```

## 设计原则

| 原则 | 说明 |
|------|------|
| 协议隔离 | 每种协议独立模块 |
| 格式统一 | 统一的数据接口 |
| 可扩展 | 易于添加新协议 |
